## Report Structure

All three reports follow the same core structure to address the three required evaluation points:

### 1. How to Run the Program
- System requirements and dependencies
- Installation and building instructions
- Running the application (graphical and text modes)
- Testing procedures
- Troubleshooting common issues

### 2. AI Implementation
- AI architecture overview (BasicAI and BotzoneAI)
- Algorithm design and implementation details
- External AI integration (bot003 from amazing-amazons project)
- Performance characteristics and decision-making process
- Configuration and technical references

### 3. Special Features
- Dual interface system (graphical and text modes)
- Comprehensive save/load system with JSON serialization
- Enhanced user experience features:
  - "Continue Previous Game" functionality
  - Non-blocking AI moves
  - Scrollable load screen
  - Modern color scheme and visual feedback
- Keyboard shortcuts and controls
- Testing and quality assurance
